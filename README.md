## Usage

1. Enter a webpage URL
2. Click "Generate"
3. Download the generated llms.txt file

## Getting Started

1. Clone the repository
2. Run `pnpm install`
3. Copy `.env.example` to `.env.local` and add your Google Gemini API key
4. Run `pnpm dev`
